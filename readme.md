\# CSTrainer: Ontology-Driven Multi-Agent AI System for Oil \& Gas Scientific Research

An open-source implementation of the paper:  

\*\*CSTrainer: An ontology driven multiple AIGC agentic approach for oil and gas scientific research supporting\*\*



\## Core Capabilities

\- Multi-model ensemble (GPT-4o, DeepSeek, Qwen, Hunyuan, Doubao-Seed)

\- Multi-agent clustering, debate, and cross-model voting

\- Dynamic distributed ontology construction

\- Lightweight human-in-the-loop (HIL) feedback

\- End-to-end pipeline: Question → Ontology → Technical Route → Executable Workflow
